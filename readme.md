#一个初学前端使用的express服务器
## 用途
做个简单脚手架吧，实验室有想用的可以试试
## 使用说明
### 环境部署
* node v6.9.1
### 初始化

1. 下载
2. 在文件夹下运行命令行`npm install` 
3. 执行`node app.js`
4. 在浏览器中打开`http://localhost:3001/`

### 目录说明
静态文件部署在src文件夹內，调用使用类似于

`<script type="text/javascript" src = './js/index.js'>`

的方式使用。



##附言
如果想处理请求数据自己看下`express`的api。
